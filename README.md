# kid-fetch

Application to fetch your kid after school as a command order.

#

# Deploy

```bash
npm pack
```

# Install

```bash
export ACCESS_TOKEN="xxx"

echo $ACCESS_TOKEN

npm install
```

## ⚠️ Https certificate depends on server IP and should be regenerated if changed using:

```bash
npm run generate-https-cert
```

# Run

```bash
npm run start
```
